# K-SIS Shortcuts

## Global Shortcuts
Save = Ctrl + S  
Save & Back = Ctrl + Shift + S or F12  
Back = Esc

## Student List Shortcuts
Filter Menu = Ctrl + F  
Enroll Student = F2  
Student Profile = F3  
Score Card Entry = R or F6  
Progress Goal = F7  
Progress History = F8  
Level Study Plan = F9  
Score Card Plan = F10  
Student Comments = F11  

## Score Card Shortcuts
Date Range = Ctrl + D  
Date Range & Save = Ctrl + Shift + D  
Save & Next Student = Alt + >  
Save & Previous Student = Alt + <
